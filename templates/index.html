Свободный VPN

Вы находитесь на сайте свободного проекта, который предоставляет бесплатный, анонимный и неограниченный по скорости VPN на основе технологии Shadowsocks. Физически сервер находится в Германии. Поддерживаются платформы Windows, Android и iOs. Для начала работы скачайте клиент для своего устройства.

Windows Android Apple

Как использовать

  1. Скачать и установить клиент для вашей операционной системы
  2. Запустить клиент и добавить новый сервер с помощью QR-кода либо с помощью ручной настройки.
  3. Для проверки включите VPN и обновите эту страницу. Смотрите раздел "Проверка"
